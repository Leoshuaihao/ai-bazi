# 第3章 六亲与大运——V2核心验证维度

## 3.1 六亲验证：十神+宫位双轨交叉证伪法

V2在V1的旺衰-格局-用神验证基础上，引入六亲维度的独立证伪。其方法论根植于命理经典的两条并行线索——十神定性与宫位定位。

《子平真诠·论六亲》确立十神配六亲法则："生我者为父母。正印为母，偏财为父……我克者为财星，乃妻妾……同我者为比劫，乃兄弟" [1]。通过十神旺衰喜忌推断六亲吉凶——用神旺相则六亲得力，忌神休囚则有亏。

《滴天髓》则强调宫位作用。《夫妻》篇指出"以日支为夫妻宫，以财官为夫妻星" [2]；《父母》篇明言"年月柱为父母宫" [3]；《兄弟》篇要求"兼看月柱——月柱为兄弟之宫位" [4]。十神与宫位的双轨制构成交叉证伪基础：双轨一致时置信度最高；矛盾时系统标记为待验证项，由用户反馈纠偏。

V2对每条六亲预测分别计算十神和宫位维度的独立得分。双轨一致则置信度+1级；双轨矛盾则降级为"推测性结论"。《三命通会·卷七》总结："六亲之论，以十神定其本，以宫位明其位，以旺衰判其力" [5]，正是双轨证伪的理论支柱。

## 3.2 大运验证：关键大运事件回溯法

大运是鉴别力最高的验证维度。每一大运十年，其间升学、就业、婚恋、父母变故等事件时间节点高度确定，几乎不存在模棱两可的解释空间。《子平真诠·论大运流年》所言"大运主十年之吉凶，流年主一岁之休咎。运为经，岁为纬" [6]，正是此鉴别力的经典表达。

V2选取3-5个关键节点回溯：

1. **第一运（约1-10岁）**：判断父母根基。《滴天髓》曰"年月柱喜安静，忌刑冲克害" [3]，父母宫被冲克时家变信息最确定。
2. **学龄运（约10-20岁）**：判断学业。印星得地则顺遂，财星坏印则受阻。
3. **婚恋运（约20-30岁）**：婚姻契机。大运引动夫妻宫或夫妻星为核心信号，"日支为配偶所居之地，五行生克直接关系婚姻质量" [2]。
4. **事业运（约30-40岁）**：职业轨迹。官星/财星表现直接反映事业升沉。
5. **父母变故窗口**：大运克财（父）或克印（母）之年。此节点鉴别力极高，因父母存亡不容歧义。

至少3个节点准确匹配才能通过大运维度验证。

## 3.3 六亲与大运联合证伪

六亲与大运交叉可大幅提升证伪精度——六亲信息在大运时间轴上演化，大运事件必然落在具体六亲关系上。

《子平真诠》指出"偏财在年柱为用神而旺相，祖上富贵，父亲有根基。偏财在月柱为用神而旺相者，父亲得力在青年" [1]。同一六亲在不同宫位的信息差异，映射到不同大运阶段的时空分布。若用户反馈"父亲20岁后才有成"，则年柱偏财被证伪、月柱偏财被证实——宫位-大运时空一致性由此获得校验。

婚姻验证同样适用：日支被冲且发生在婚恋运——双信号叠加，婚姻波动置信度极高。若日支被冲但婚恋运平稳，则标记为"弱信号"。

《三命通会》直言"以年为祖业，月为父母兄弟门户，日为妻妾己身，时为子息。须看四柱之中父母兄弟妻子星具何地" [5]，天然蕴含宫位→六亲星→旺衰阶段的三元交叉，为V2联合证伪的理论基石。

## 3.4 V2增强与V1对比——四维度到六维度覆盖率量化

V1验证体系覆盖旺衰、格局、用神、十神四维度，只能验证"命盘本身对不对"，无法验证"命盘在时间轴上的表现对不对"。V2引入六亲和大运两个增量维度：

| 维度 | V1 | V2 | 增量意义 |
|------|:--:|:--:|------|
| 旺衰验证 | ✓ | ✓ | 日主强弱 |
| 格局验证 | ✓ | ✓ | 月令定型 |
| 用神验证 | ✓ | ✓ | 喜忌选取 |
| 十神解读 | ✓ | ✓ | 含义释义 |
| **六亲验证** | ✗ | **✓** | 十神+宫位双轨证伪 |
| **大运验证** | ✗ | **✓** | 关键节点回溯 |

六亲维度使验证从"结构性验证"扩展到"关系性验证"——用户可逐条确认六亲推断，而非仅评判抽象命理概念。大运维度使验证从"静态命盘"升级为"动态轨迹"——"运到用神，方为得助" [6] 的经典命题得以用实际人生数据量化。

V1中四条维度独立运作，无法交叉验证；V2中维度间形成证伪网络：六亲失败可倒查旺衰环节，大运偏差可暴露用神问题——错误定位从"某阶段"精进至"某阶段×某典籍"。

覆盖率量化：V1四维度覆盖约60%的用户反馈场景，无法处理六亲关系和人生时间线约40%场景；V2六维度将覆盖率提至约85%以上，剩余盲区主要存在于流月/流日尺度和极端命例。
